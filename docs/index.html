La Maison

Contexte

Ce projet a été créé dans le cadre d’un exercice de conception multimédia dans un cours au Cégep montmorency dans un technique d'intégration multimedia . L’objectif est de plonger le joueur dans une atmosphère mystérieuse où ses choix détermineront la suite de l’histoire.

La page d’accueil vous présente le contexte général et vous permet de lancer la partie.
